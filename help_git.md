# Инструкция по командам GIT

## Создание репозитория  
```sh
git init
```
## Добавление файла для индексации 
```sh
git add <filename>
```
## Запись изменений в репозиторий 
```sh
git commit -m "Message"
```
## Показать лог изменений 
```sh
git log
```
## Показать лог изменений в одну строку
```sh
git log --oneline
```
